Step 5: Set Up MySQL Database

Log in and create database user:

# Log into MySQL as the root user
sudo mysql

At the mysql> prompt, run:

-- Create a new user for our application
CREATE USER 'hw5_user'@'localhost' IDENTIFIED BY 'Password123!';

-- Grant this user privileges on the 'products' database
GRANT ALL PRIVILEGES ON products.* TO 'hw5_user'@'localhost';

-- Apply the changes and exit
FLUSH PRIVILEGES;
exit;

Run SQL script to set up DB:

# Navigate to your project directory
cd /var/www/html/ch19

# Execute the SQL script to create and populate the database
sudo mysql < Products.sql
Security Note: In production, use a stronger password and limit privileges
