Step 4: Prepare Web Directory & Upload Files

Add user to the Apache web group:

# Add your user to the www-data group
sudo usermod -a -G www-data ubuntu

IMPORTANT:

For this change to take effect, you must log out (exit) and log back in with SSH.

After reconnecting, set the correct permissions:

# Change ownership of the web directory
sudo chown -R ubuntu:www-data /var/www/html

# Set directory permissions to be writable by the group
sudo chmod -R 775 /var/www/html

Upload your project files from your local machine:

# On your LOCAL machine, use scp to upload the folder
scp -r -i /path/to/your/my-aws-key.pem ch19 ubuntu@your-ec2-public-ip:/var/www/html/
