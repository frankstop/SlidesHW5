Step 1.2: Security Group & Network Settings

Configure your network settings:

Edit security group:
Add rule: SSH (source: 'My IP')

Allows only your computer to connect securely via command line

Add rule: HTTP (source: 'Anywhere' or 0.0.0.0/0)

Allows anyone to view your website

Review and launch instance
Wait for server to start (~1-2 min)

Security Note: Restricting SSH access to your IP only is a security best practice
