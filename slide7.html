Step 3: Install the LAMP Stack

Update and install Apache, MySQL, PHP:

# Update package lists
sudo apt update

# Install Apache, MySQL Server, and PHP
sudo apt install apache2 mysql-server php -y

Install PHP modules for Apache and MySQL connectivity:

# Install essential PHP modules
sudo apt install libapache2-mod-php php-mysql -y

Restart Apache to apply changes:

# Restart Apache to apply changes
sudo systemctl restart apache2
You can verify installation success by checking service status:
systemctl status apache2 and systemctl status mysql
